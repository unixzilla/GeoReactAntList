## Requirement

Node.JS


## Installation
Open terminal, git clone the project
```sh
git clone https://github.com/unixzilla/GeoReactAntList.git
```

then

```sh
cd GeoReactAntList
```

then


```sh
npm install
```

then


```sh
npm start
```

If you want to stop, Press Control+C


Open your browser goto http://localhost:3000




enjoy :)
